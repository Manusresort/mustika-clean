# Prompts — Orientation

Deze map bevat prompts voor sandbox, pilots en experimentele agents.
Ze beschrijven GEDRAG, maar vormen geen beleid.

Guidelines:
- sandbox/prompts ≠ productiebeleid
- wijzigingen zijn documentair, niet automatisch beslissend
- bij twijfel: raadpleeg CANONICAL_INDEX.md en manifest_p5.yaml

Doel: nieuwe lezers helpen begrijpen waar prompts in het grotere
plaatje passen, zonder structuur te wijzigen.
