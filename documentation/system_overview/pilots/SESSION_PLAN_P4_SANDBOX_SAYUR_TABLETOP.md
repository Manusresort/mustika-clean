Title: Table-Top Session Plan — SAYUR Sandbox Rehearsal

Status:
PLANNING DOCUMENT — NOT EXECUTION AUTHORIZATION

SessionPurpose:
- Een volledige rehearsal uitvoeren volgens het runbook.
- Controleren of rollen, soft-stops en logging duidelijk werken.
- Vastleggen waar het proces onduidelijk is — niet oplossen.

SessionScope:
- Alleen SAJUR-A + topregels SAJUR-B
- Geen nieuwe bronnen, geen publicatie, geen glossary-besluiten.

Duration:
60–75 minuten totaal.

Participants & Roles:
- Facilitator (acts as Orchestrator)
- Translation (human)
- Annotation (human)
- Challenger
- Methodology Archivist
- Optional Observer (notes usability + frictie)

Prerequisites:
- Toegang tot: excerpt file, EXCERPT_MAP, pilotplan, runbook
- Gedeelde notitieruimte voor logs en soft-stops
- Duidelijke afspraak: “documenteren > oplossen”

Agenda (with timing):

0–5 min — Opening
- Doel herhalen
- Stop-model uitleggen
- Rollen bevestigen

5–20 min — Translation (proposal-only)
- Vertaling met [UNCERTAIN] tags
- EvidenceRef noteren

20–35 min — Annotation
- HISTORICAL / GLOSSARY / SAFETY / OCR labels
- Geen normalisatie

35–45 min — Challenger
- Impliciete beslissingen markeren
- SOFT-STOPS plaatsen waar lezer-betekenis kan verschuiven

45–55 min — Orchestrator + Archivist
- Triage: documenteren vs escaleren
- Log registreren met LOG FIELDS template

55–70 min — Debrief
- Wat werkte / wat brak
- Welke regels/velden misten
- Noteer “improvement ideas” (proposal-only)

Materials:
- RUNBOOK_P4_SANDBOX_SAYUR_TABLETOP.md
- PILOT_TRANSLATION_PLAN_SAYUR
- CHAPTER_PREP_REPORT (Sayur)
- EXCERPT_MAP + excerpt file

Explicit “Do-Not” List:
- Geen runtime-agents
- Geen glossary-besluiten
- Geen inhoud normaliseren
- Niets publiceren of verplaatsen
- Geen on-the-fly tool aanpassingen

ExpectedOutputs:
- Rehearsal log
- Soft-stop lijst
- Incidenten/ambiguiteit notities
- Korte AFTER-ACTION NOTE (draft)

SuccessSignals:
- Soft-stops werden bewust gebruikt
- Elke stap herleidbaar naar EvidenceRef
- Geen beslissingen vermomd als annotaties
- Rollback blijft triviaal

FailureSignals:
- Soft-stops genegeerd
- Rationale niet gelogd
- Deelnemers onzeker over escalatiepad

NextStep (after session):
- Schrijf AFTER-ACTION NOTE
- Vertaal bevindingen naar verbeteringen in runbook/pilots
- Geen runtime totdat governance bevestigt.

Status reminder:
Dit plan organiseert een oefening.
Het geeft GEEN toestemming voor echte uitvoering.
