# BRON_LACUNES — Mustika Rasa (Editie A)

## Doel van dit document

Dit document registreert waargenomen lacunes, onzekerheden en mogelijke
discrepanties tussen de digitale bron (scan/PDF/OCR) en het fysieke boek
*Mustika Rasa*.

Het document:
- bevat **geen reconstructies**;
- bevat **geen correcties van de brontekst**;
- fungeert uitsluitend als signalerings- en verwijzingslijst.

Alle inhoudelijke beslissingen blijven expliciet elders vastgelegd.
