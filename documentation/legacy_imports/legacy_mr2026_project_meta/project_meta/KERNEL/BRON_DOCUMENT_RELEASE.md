## Aangewezen brondocument
De canonieke bron voor Editie A is:
- Tekstbasis: step3_canonical/page-*.txt (1214 bestanden)

Deze tekst vormt de vaste referentie voor alle verdere bewerkingen
(annotatie, vertaling, publicatie) binnen Editie A.

## Totstandkoming
De tekst is tot stand gekomen via:
- OCR van gescand bronmateriaal
- OCR-vetting (detectie van systematische fouten)
- Structurele en mechanische extractie (Stap 4/5)
- Annotaties (Stap 6)
- LLM-gestuurde lacunedetectie (Stap 3b)

## Bekende beperkingen
- De tekst bevat onopgeloste OCR-artefacten.
- Mogelijk ontbreken pagina’s of tekstsegmenten.
- De volgorde van pagina’s kan op enkele plaatsen onzeker zijn.
- Er is afgezien van handmatige verificatie per signaal.

Deze beperkingen zijn expliciet gedocumenteerd in:
- project_meta/SCAN_SIGNALS/
- project_meta/BRON_LACUNES.md

## Methodologische keuze
Er is bewust gekozen voor transparantie boven reconstructie.
Onzekerheden zijn zichtbaar gelaten en niet stilzwijgend gecorrigeerd.
